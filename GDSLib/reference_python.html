API Reference

ElementStructures

PolygonSet

class ElementStructures.PolygonSet

基类：object

PolygenSet 类.

该类为器件定义的基础类，其中仅存储定义器件的多边形数据，多边形所处layer和datatype与之在index上需一一对应.

变量:

  • polygons (list of array-like[N][2]) – 多边形定点XY坐标的列表.

  • layers (list of int) – 多边形所处layer的列表

  • datatypes (list of int) – 多边形所在datatype的列表

generate_pcell(name: str)

根据当前所存储的多边形, 以及对应layer和datatype, 生成 gdstk.PCell, 用于进一步放置更高层次的Cell中.

参数:

name (str) – PCell的名字

返回:

out – 放置当前PolygonSet所存储多边形的PCell.

返回类型:

gdstk.Pcell

Path

class ElementStructures.Path(start, width, layer=None, datatype=None, precision=0.001)

基类：PolygonSet

Path 类.

用于 gdstk 与此前基于 gdspy 所构建器件库间的兼容性适配.

变量:

  • path (gdstk.FlexPath) – gdstk包所提供的的FlexPath对象，如果对gdstk包十分熟悉，可以直接调用gdstk提供的方法.

  • endAngle (double) – path末端切线的方向，在类初始化时默认取为0, 随后会根据调用生成路径的具体方法，自动更新.

参数:

  • start (array-like[1][2]) – 路径起始点的坐标.

  • width (double or array-like[1][N]) – 路径的宽度，该参数可以给成列表，若如此做，则会生成多条完全重合，而宽度不同的路径. 该feature当需要生成与路径重合的heater时，会较为便捷.

  • layer (array-like[1][N]) – 放置路径的layer，与宽度定义一一对应，若取为默认值None，则会放置于layer 0.

  • datatype (array-like[1][N]) – datatype，与宽度定义一一对应，若取为默认值None，则datatype = 0.

  • precision (double) – path图形的取样精度，默认为1e-3.

segment(length, width=None, direction=None)

将一截直线段加入 self.path.

参数:

  • length (int) – 线段长度

  • width (double or array-like[1][N]) – 线段的末端宽度（起始宽度以上一段路径结束时为准，无法手动干预），可用于绘制taper

  • direction (double) – 线段的方向，默认为None，此时方向取为 self.endAngle

segments(points, width=None, join='natural', end='flush')

将多个线段加入 self.path

参数:

  • points (array-like[N][2]) – 线段终点相对于起始点的XY坐标

  • width (double or array-like[1][N]) – 线段的末端宽度（起始宽度以上一段路径结束时为准，无法手动干预），可用于绘制taper

  • join (str) – 线段的连接方式，见gdstk.FlexPath文档

  • end (str) – 线段的端面形状，见gdstk.FlexPath文档，此参数一般无需修改，取默认值即可

turn(radius, angle, width=None)

将路径转弯加入 self.path

参数:

  • radius (double) – 半径

  • angle (double) – 转角，向左为正，向右为负

  • width (double or array-like[1][N]) – 线段的末端宽度（起始宽度以上一段路径结束时为准，无法手动干预）

add_current_path()

将当前self.path转为多边形XY坐标，存入 self.polygons，不推荐用户直接调用.

end_current_path(start=None, width=None, layer=None, datatype=None, precision=0.001)

将当前self.path转为多边形XY坐标，存入 self.polygons，若start非空，则初始化一条新的路径.

参数:

  • start (array-like[1][2]) – 新路径起始点的坐标.

  • width (double or array-like[1][N]) – 新路径的宽度，该参数可以给成列表，若如此做，则会生成多条完全重合，而宽度不同的路径. 该feature当需要生成与路径重合的heater时，会较为便捷.

  • layer (array-like[1][N]) – 放置新路径的layer，与宽度定义一一对应，若取为默认值None，则会放置于layer 0.

  • datatype (array-like[1][N]) – datatype，与宽度定义一一对应，若取为默认值None，则datatype = 0.

  • precision (double) – path图形的取样精度，默认为1e-3.

generate_pcell(name: str)

根据当前所存储的多边形, 以及对应layer和datatype, 生成 gdstk.PCell, 用于进一步放置更高层次的Cell中.

参数:

name (str) – PCell的名字

返回:

out – 放置当前PolygonSet所存储多边形的PCell.

返回类型:

gdstk.Pcell

Waveguide

class ElementStructures.Waveguide(start, width, layer=0, datatype=0, precision=0.001)

基类：Path

Waveguide 类.

基于Path类构建，额外提供了taper构建方法，以及基于阿基米德螺旋线的波导构建方法。

变量:

  • total_length_list (list of double) – 存储每根波导的总长.

  • taperLength (double) – 用于定义taper，为过渡至taper部分长度.

  • taperWidth (double) – 用于定义taper，为taper部分宽度

  • endLength (double) – 用于定义taper，为波导收窄后伸出部分的长度.

参数:

  • start (array-like[1][2]) – 路径起始点的坐标.

  • width (double or array-like[1][N]) – 路径的宽度，该参数可以给成列表，若如此做，则会生成多条完全重合，而宽度不同的路径. 该feature当需要生成与路径重合的heater时，会较为便捷.

  • layer (array-like[1][N]) – 放置路径的layer，与宽度定义一一对应，若取为默认值None，则会放置于layer 0.

  • datatype (array-like[1][N]) – datatype，与宽度定义一一对应，若取为默认值None，则datatype = 0.

  • precision (double) – path图形的取样精度，默认为1e-3.

segment(length, width=None, direction=None)

将一截直线段加入 self.path.

参数:

spline(ncp, symm=False)

生成欧拉曲线，曲线起点会自动采用Waveguide的末端点坐标、切向角度以及曲率，末端由ncp给出.

参数:

  • ncp (array-like[1][4]) – ncp包含定义该曲线的四个参数：dx, dy, 末端角度[rad], 末端曲率.

  • symm (boolean) – 是否强制曲线关于中心点对称.

将当前self.path转为多边形XY坐标，存入 self.polygons，若start非空，则初始化一条新的路径.
